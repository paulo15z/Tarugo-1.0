# Tarugo Frontend Design Guidelines

**Estilo principal**: Industrial Clean

**Cores principais**:
- Background: #0f0f0f (dark)
- Primary: #3b82f6 (azul industrial)
- Success: #22c55e
- Warning: #eab308

**Tipografia**:
- Headings: Fonte com personalidade (Satoshi / Space Grotesk / similar)
- Body: Inter ou system sans-serif

**Princípios**:
- Legibilidade em primeiro lugar (fábrica)
- Contraste alto
- Cards grandes para progresso de lotes/módulos
- Feedback claro na bipagem (sucesso/erro com animação sutil)

Use sempre o skill `tarugo-frontend-design` quando criar ou melhorar telas.
